The main focus of this project was to practice using 
- Unordered lists ('<ul'>)
-Ordered lists ('<ol>')
## What I did
Started with the basic HTML boilerplate setup
Added headings using <h1>, <h2>, and <h3>.
Includded paragraph text using the <p> element.
## Notes
This project uses only **HTML**.
There is **there no styling** (CSS) or **interactivity** (Javascript).
